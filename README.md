# Esther Nnamah - Portfolio

A modern, professional portfolio website for Esther Nnamah, a Brand Strategist & Content Creator.

## Features

- **Modern Design**: Deep blue and white color scheme with smooth animations
- **Responsive**: Fully responsive design that works on all devices
- **AI Chat**: Integrated Claude AI chatbot for visitor engagement
- **Social Media Integration**: Links to Instagram, TikTok, and Facebook
- **Portfolio Section**: Showcase of featured work and content campaigns
- **Services Overview**: Detailed breakdown of services offered
- **Contact Form**: Easy way for potential clients to get in touch

## Tech Stack

- **React 19**: Latest React version
- **Vite**: Fast build tool and dev server
- **CSS3**: Modern styling with animations and gradients
- **Claude API**: AI-powered chatbot integration

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Set up environment variables:
Create a `.env.local` file and add your Claude API key:
```
VITE_CLAUDE_API_KEY=your_api_key_here
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

## Build

To build for production:
```bash
npm run build
```

## Preview

To preview the production build locally:
```bash
npm run preview
```

## Customization

### Colors

The portfolio uses a deep blue and white color scheme. To customize:
- Primary blue: `#003d82`
- Secondary blue: `#0066cc`
- Background: `#ffffff`, `#f5f8fc`, `#f9fbfd`

Edit `src/App.css` to change colors globally.

### Content

Edit `src/App.jsx` to customize:
- Personal information
- Services and expertise
- Portfolio projects
- Social media links
- Contact information
