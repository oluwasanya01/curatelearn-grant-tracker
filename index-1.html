⚡ Zapier Auto-Sync Setup

1
Create a Google Sheet with these exact column headers in row 1:
Grant Name · Sector · Amount · App No · Date Applied · Deadline · Status · Link · Notes · Next Action
2
Publish the sheet as CSV: File → Share → Publish to web → Entire Document → CSV → Publish. Copy the URL and paste it above.
3
Set up Zapier Zap: Trigger = Gmail (new email matching grant OR awarded OR "not selected") → Action = Google Sheets (add row). Map: Subject → Grant Name, Date → Date Applied, Snippet → Notes.
4
Click Sync anytime (or reload the page) to pull the latest data from your sheet. Everyone with this link sees the same live data! 🎉
